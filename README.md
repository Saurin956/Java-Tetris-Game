# Java Tetris Game

A Tetris-style game developed in Java using Object-Oriented Programming (OOP) concepts. The game features different block shapes, collision detection, row clearing mechanics, score tracking, and game state management.

## Features

- Multiple Tetris block shapes (I, J, L, O, S, T, Z)
- Block movement and rotation
- Collision detection with walls and locked blocks
- Row completion and clearing system
- Score tracking
- Game over detection
- Resource management using external assets
- Save and configuration file support

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Eclipse IDE
- Git & GitHub

## Project Structure

```
Game/
├── src/        # Java source code
├── res/        # Game assets and resources
├── config.txt  # Configuration settings
└── save.dat    # Save data
```

## OOP Concepts Used

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Saurin956/Java-Tetris-Game.git
```

2. Import the project into Eclipse.

3. Run the main Java file to start the game.
